# Titanic Data Cleaning and Preprocessing Project

## Problem Statement
The objective of this project is to clean and preprocess the Titanic dataset by handling missing values, removing inconsistencies, and preparing the data for further analysis.

## Dataset Details
Dataset: Titanic Dataset

The dataset contains passenger information such as:
- Passenger ID
- Name
- Age
- Gender
- Ticket details
- Fare
- Survival status

## Approach
The following steps were performed:

- Loaded the Titanic dataset
- Checked missing values and data types
- Handled missing data
- Removed unnecessary columns
- Cleaned and prepared the dataset
- Saved the cleaned dataset for further analysis

## Results
- Successfully cleaned the Titanic dataset
- Improved data quality by handling missing values and inconsistencies
- Created a cleaned dataset ready for analysis

## Files Included
- Titanic_Cleaning.ipynb - Data cleaning notebook
- Titanic Dataset.csv - Original dataset
- Cleaned_Titanic.csv - Cleaned dataset
